# CobaltEssentials
CobaltEssentials Plugin For The BeamMP Mod
# Commands
* /help - Get a list of each commmand
* /list - Get a list of each player and their ID
* /kick [ID] - Kick a player from the session
* /ban [ID] - Ban a player from the session
